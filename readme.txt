Add a link to web-whatsapp chat on each product, with information about the product. It allows asign the work schedule up to 2 employees.

== Description ==
Integrate WhatsApp support dialog directly into your Woocommerce store and that will help you to increase leads/sales from your website. this is well design with lightwight code help to attract users for chat with active agent using WhatApp application.

== Installation ==

= Installation =
1. In your WordPress admin panel, go to Plugins > New Plugin
2. Find the product-support-now plugin by Frank Ortiz and click Install now
3. Alternatively, download the plugin and upload the contents of product-support-now.zip to your plugins directory, which usually is /wp-content/plugins/
4. Activate the plugin
5. Enjoy
